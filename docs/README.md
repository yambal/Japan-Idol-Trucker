# JPN Idol Trucker

![icon](https://github.com/yambal/Japan-Idol-Trucker/raw/master/icon.jpg)

## 概要
Mod for **Euro Truck Simulator 2**

- ドライバーをアイドルにします
  - 標準ドライバー（341人まで）は性別を維持し、これまでのゲームデータでも性転換が発生しません
  - 拡張追加ドライバーについては性別を留意しません
